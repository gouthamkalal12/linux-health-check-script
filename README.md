# Linux Server Health Check Script

## Overview

This project contains a Bash script used to perform basic Linux server health monitoring and troubleshooting.

The script collects important system information such as:

- Hostname
- System Uptime
- Memory Usage
- Disk Usage
- Top CPU Consuming Processes
- Top Memory Consuming Processes

## Technologies Used

- Linux
- Bash Shell Scripting

## How to Run

Make the script executable:

```bash
chmod +x health-check.sh
```

Run the script:

```bash
./health-check.sh
```

## Sample Output

The script displays:

- Server Hostname
- System Uptime
- Memory Utilization
- Disk Utilization
- CPU Usage Statistics
- Running Processes
